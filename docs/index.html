Features

Everything you need. Nothing you don't.

Built from scratch to replace your OS file manager with something faster and smarter.

📂
Column browsing
Navigate deep folder trees side-by-side with smooth horizontal scrolling. Select items and preview instantly without switching context.
🏷️
Custom tags
Create colour-coded tags and assign them to any file or folder. Filter your entire file system by tag in one click.
⚡
Instant search
Recursive full-filesystem search finds files as you type, scanning hundreds of folders in seconds without indexing.
🗂️
Smart folders
Built-in smart views for Recent, All Images, Large Files, Screenshots, and Documents — always up to date automatically.
👁️
Rich previews
Preview images, videos, audio waveforms, text files, PDFs, and documents directly in the panel without opening another app.
🎨
Beautiful design
Crafted with care — glassmorphic UI, smooth animations, accent colour themes, and a custom frameless window that feels at home on any OS.
